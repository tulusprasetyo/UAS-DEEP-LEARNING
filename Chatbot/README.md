🎓 Chatbot Jadwal Kuliah Semester Ganjil TA 2025/2026

Aplikasi Chatbot berbasis Web menggunakan **Streamlit** dan **NLTK** untuk menjawab pertanyaan seputar jadwal kuliah berdasarkan hari, dosen, semester, atau ruangan.

---

## 🚀 Cara Menjalankan Aplikasi Demo

Ikuti langkah-langkah di bawah ini untuk menjalankan aplikasi ini di lokal laptop Anda:

### 1. Clone Repositori

Clone project ini terlebih dahulu ke komputer Anda:

```bash
git clone [https://github.com/tulusprasetyo/UAS-DEEP-LEARNING.git](https://github.com/tulusprasetyo/UAS-DEEP-LEARNING.git)
cd UAS-DEEP-LEARNING

2. Masuk ke Folder Inti Chatbot

Karena repositori ini berbentuk monorepo dan struktur kodenya berada di dalam sub-folder, silakan masuk ke direktori tempat file utama berada:
Bash

cd Chatbot/chatbot

3. ... Lanjutan Langkah (Optional)

Jika Anda menggunakan Virtual Environment (venv), silakan aktifkan terlebih dahulu sebelum lanjut ke langkah berikutnya.
4. Instalasi Library

Pastikan Python sudah terinstal di komputer Anda, lalu jalankan perintah berikut untuk menginstal semua library yang dibutuhkan secara manual:
Bash

pip install streamlit nltk torch pandas

5. Jalankan Aplikasi

Setelah semua komponen terinstal, jalankan aplikasi menggunakan nama file yang sesuai:
Bash

streamlit run chatbot_app.py

Aplikasi otomatis akan terbuka di browser Anda pada alamat http://localhost:8501.
```
